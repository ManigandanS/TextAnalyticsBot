What is Text Analytics Bot

Text Analytics Bot is built using Microsoft Bot Framework and it's powered by Language Understanding Intelligent Service (LUIS) and Text Analytics API which is available with Microsoft Cognitive Services.

You can initiate the dialog by greeting to the bot. As this bot is still learning his stuff out, currently you can only submit feedback for an event, forum etc. You can trigger that intent by sending something like "submit feedback, share feedback etc".

Please do use this bot and help him train himself. And of course, your contributions are most welcome to improve his functionality. Join now.
